1. How does javascript works?

JavaScript একটি scripting language। এটি একটি interpreted language, যার মানে এটির কম্পাইলারের প্রয়োজন নেই। JavaScript কোড সরাসরি একটি ওয়েব ব্রাউজারে চলে। JavaScript ইঞ্জিন script পড়ে। তারপর এটি মেশিন ভাষায় script কম্পাইল করে এবং তারপর মেশিন কোড রান করে। JavaScript dynamic ভাবে ওয়েব পৃষ্ঠাগুলির উপাদান নিয়ন্ত্রণ করে। এটি ওয়েব ব্রাউজার, API এবং ওয়েব সার্ভারেও কাজ করে।

2. Difference between localStorage and sessionStorage?

  • LocalStorage a stored ডেটা ব্রাউজারে সেশনে সংরক্ষণ করা হয়। তাই domain a নতুন ট্যাবে ওপেন করা হলেও বা ব্রাউজার বন্ধ করে পুনরায় খোলা হলেও storage data সেভ হয়ে থাকে।
  • পেজ end করলে sessionStorage cleared হয়ে যায়। একটি পেজ যতক্ষণ ট্যাব বা ব্রাউজারে খোলা থাকে sessionStorage ততক্ষণ স্থায়ী হয়ে থাকে। যখন আমরা একটি নতুন ট্যাব বা একটি নতুন উইন্ডো খুলি, এটি তখন একটি নতুন সেশন এবং নতুন storage তৈরি করে।
